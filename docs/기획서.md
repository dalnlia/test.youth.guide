# 내용 상세

## 1. 콘셉트

- 8개의 자취 일상 질문을 통해 유저 스스로 자신의 자취 성향 유형을 진단하고, 결과에 따라 딱 맞는 청년주택 유형을 자연스럽게 추천받는 인터랙티브 설문
    
    ![image.png](image.png)
    

### (1) 유형 및 질문 로직

- 고방 [청년주택 유형 알아보기](https://gobang.kr/youth/notices/guide) 속 8가지 주택 유형을 ‘자취 유형’으로 치환
- 설문 응답별 점수 +1점 집계 후 가장 높은 점수를 받은 유형을 최종 결과로 노출

![image.png](image%201.png)

![image.png](image%202.png)

![image.png](image%203.png)

### (2) 결과 화면 CTA

- 결과 화면 내 CTA 총 4개 삽입 → 앱스스크립트를 통해 메인, 서브1, 서브2 추적
    - 메인 [나한테 딱 맞는 청년주택 확인] → 유형별 가이드 페이지로 이동
    - 서브1 [테스트 공유하기] → 링크 복사
    - 서브2 [이미지 저장하기] → 4:5 PNG 저장
    - [다시 테스트하기] - 처음으로 이동
- 유저ID 기준: 세션스토리지(브라우저 탭 세션 단위)
    - 필요 시 로컬스토리지로 변경 예정

# 기대효과

## (1) 정량적 목표

- **지표1**: 유입CTR `청년주택 확인 클릭수 / 테스트 완료수` 5%
- **지표2**: 공유CTR `공유 클릭수 / 테스트 완료수` 5%

## (2) 정성적 목표

- **공유 바이럴**: 무슨 유형 나왔는지 공유가 쉬워 바이럴 가능성 상
- **자연스러운 유입**: 광고가 아닌 콘텐츠로 접근하여 거부감 없는 서비스 노출
- **타깃 정확도**: 자취러가 직접 참여하는 구조로 청년주택 실수요자에게 정확히 닿음
- **브랜드 친밀도 상승**: 딱딱한 청년주택 정보를 재미있게 전달하며 브랜드 호감도 제고

# 개선사항

## (1) 콘텐츠 개선

- 점수 집계 방식: 동점 시 먼저 정의된 유형으로 뜸 → 질문 수를 조절하거나 로직 일부 개선 필요함
- 전반적 리라이팅: ‘숨만쉬어도적자형’과 같이 다소 부정적인 표현을 비롯한 전반적인 리라이팅 필요
- 디자인 영역 개선: 레이아웃, 테스트시작/결과 페이지의 이미지 등 디자인 영역의 개선 (ChatGPT)
- OG 이미지 커스텀: 링크 공유 시 미리보기 이미지 설정 필요

## (2) 데이터 집계 개선

- 현재는 CTA 버튼 3건의 클릭수만 집계할 수 있음 → 테스트 완료 시점에도 로그 추가하여 [공유 클릭수/테스트 완료수] 확인할 수 있도록 수정
